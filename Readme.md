# Smart Local Services Marketplace

A full-stack hyperlocal service booking platform where users can discover, book, and pay for nearby service providers such as electricians, plumbers, tutors, cleaners, and technicians in real time.

Built with React, Django REST Framework, PostgreSQL, Google Maps API, and Razorpay, the platform includes location-based discovery, real-time booking, secure payments, provider dashboards, ratings & reviews, notifications, and admin analytics.

## Features
- JWT Authentication
- Role-based User & Provider System
- Location-Based Service Discovery
- Real-Time Booking System
- Online Payments Integration
- Ratings & Reviews
- Provider Dashboard
- Admin Analytics Panel
- Notifications System
- Scalable SaaS Architecture

## Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Django, Django REST Framework
- Database: PostgreSQL
- APIs: Google Maps API, Razorpay, Firebase
